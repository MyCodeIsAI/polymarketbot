How to get a free Polygon RPC URL

Recommended: Alchemy (Free tier: 300M compute units/month)

  1. Go to dashboard.alchemy.com/signup
  2. Create a free account (no credit card required)
  3. Click "Create new app"
  4. Select "Polygon" as the chain and "Mainnet" as network
  5. Give it a name like "PolymarketBot"
  6. Click "View Key" on your new app
  7. Copy the WebSocket URL (starts with wss://)
  8. Paste it in the field above

Example URL format:

wss://polygon-mainnet.g.alchemy.com/v2/abc123xyz789...

Alternatives: Infura, QuickNode, or any Polygon WebSocket RPC provider. The free tier on Alchemy is more than enough for copy trading.

Environment Variables

You can also set the RPC URL via environment variable:

POLYGON_RPC_URL=wss://polygon-mainnet.g.alchemy.com/v2/YOUR_KEY

Add this to your .env file or set it in your shell before running the bot.
